What are the differences between relative, absolute, and fixed positioning?

I will attempt to explain and illustrate the differences between the aforementioned property values which define the positioning of elements.


position: static;

Elements are rendered according to their order in the document flow. This is the default value.

Static

position: relative;

Elements are rendered relative to their default Static position. The position of their default Static state is still reserved to the element, i.e. a successive in-line block does not take its pre-rendered position.

position: absolute;

Elements are removed from their parent containing element and moves upwards along it's element ancestry tree until it's encapsulated by a positioned, non-Static element. In the example below, the element with position property value ABSOLUTE is removed from it's parent Static element into the nearest ancestral positioned element which in this case is labelled GRANGPARENT. The element is now positioned relative to its GRANDPARENT element.

position: sticky;

The default Sticky setting positions the elements at a fixed position on the viewport once the user has scrolled down sufficiently enough for the element to reach the top of the viewport.
